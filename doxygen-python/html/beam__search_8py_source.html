14  """
15  Class generalizing forward beam search for seq2seq models.
16 
17  Also provides types to specify the recurrent structure of decoding:
18 
19  StateConfig:
20  initial_value: blob providing value of state at first step_model
21  state_prev_link: LinkConfig describing how recurrent step receives

23  state_link: LinkConfig describing how step writes (produces new state)

25 
26  LinkConfig:

28  offset: offset from beginning of global blob for link in time dimension
